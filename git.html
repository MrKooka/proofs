Основные
  • git add * - добавить все файлы в index
  • git commit -m'comment' - Коммит файлов из индекса
  • git status - Показать статус файлов в текущем каталоге
  • git diff - показать разницу между коммитами
  • git log - показать историю коммитов
  • git pull - получить изменения с удаленного репозитория и слить данные с тем что уже есть на локальной машине
  • git fetch - получает изменения с удаленного репозитория, но не сливает из с тем что уже есть на локальной машине
  • git push - Отправить свои изменения на удаленный репозиторий
Связанные с етками
  • git branch [название ветки] - создать ветку
    • git branch -b [название ветки] - создать ветку и сразу перейти на нее
    • git branch -d [название ветки] - удалить ветку
  • git checkout [название ветки] - Переключится на ветку
