Работа с файловой системой:
  • pwd - показать текущую директорию
  • cd - Переход по директориям
  • cat - показать содержимое файла
  • ls - показать содержимое текущей папки
  • mkdir - Создать директорию
  • mv - мереместить файла в или переименовать
  • cp - сопировать в
  • touch - создать файл
  • grep команда:
    ls | grep image
    выведет все файлы в текущей директори,которые имеют в своем названии слово image
